API Endpoints

GET /api/v1/conflicts
Get all conflicts
GET /api/v1/conflicts?status=ACTIVE
Get conflicts by status (ACTIVE, FROZEN, ENDED)
GET /api/v1/conflicts/{id}
Get conflict by ID
GET /api/v1/countries/{code}/conflicts
Get conflicts by country code (ex: UKR, RUS)
POST /api/v1/conflicts
Create new conflict
PUT /api/v1/conflicts/{id}
Update conflict
DELETE /api/v1/conflicts/{id}
Delete conflict
GET /api/v1/countries
Get all countries
GET /api/v1/factions
Get all factions
GET /api/v1/events
Get all events
